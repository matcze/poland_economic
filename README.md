# 🌍 World Economy Data Analysis

This project explores global economic trends using GDP and GDP per capita data. The analysis focuses on comparing economic growth between countries and regions, with particular attention to Poland’s economic development over time.

Using Python data analysis tools, the project visualizes how economies evolve, highlights disparities between countries, and examines how economic indicators change across different periods.

The results are presented through a series of graphs and visualizations, each followed by a short conclusion explaining the key insights. This structure keeps the analysis clear and easy to follow despite the large number of charts.

# 📊 Research Questions
This project investigates the following topics:
Poland's Economic Development
- GDP per capita growth in Poland over the years
- Poland’s total GDP growth over time
- Comparison of Poland’s economic growth during governments led by PO and PiS
International Comparisons
- GDP per capita comparison between Poland, Germany, and France
- GDP per capita difference between Poland and Germany
- GDP per capita comparison between USA and China over time
European Economy
- GDP per capita in the Euro Area in 2004 vs 2024
- Comparison of GDP among Euro Area countries in 2023 and 2024
- Global Perspective
- The 10 wealthiest countries in the world
- The 10 poorest countries in the world
- GDP growth in 2024

# 🛠 Technologies Used
The project was developed in Python using the following libraries:
    > NumPy – numerical computing
    > Pandas – data analysis and data manipulation
    > Matplotlib – data visualization
    > Seaborn – statistical data visualization

# 📈 Visualization Approach
Due to the large number of charts generated in this project, each visualization includes a short conclusion directly below the graph. This approach:
    > improves readability
    > keeps interpretations close to the data
    > helps maintain clarity throughout the analysis

# 🎯 Project Goal
The purpose of this project is to:
    > demonstrate data analysis skills
    > practice economic data visualization
    > explore global economic inequalities and trends
    > present complex data in a clear and accessible way
